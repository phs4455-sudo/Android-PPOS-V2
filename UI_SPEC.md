# UI_SPEC.md

## 공통
- 상단 고정 탑바: 좌 "THE HYUNDAI" + 날짜/시간, 우에 [점검][조회][영수증 재출력][더보기]
- 기본 레이아웃은 데스크톱/태블릿 가로 기준(2-pane)

## Restaurant(테이블)
- 상단: Area 탭 (식당가 1층 홀/룸/2층 홀/야외 테라스)
- 본문: 좌(작업영역)=테이블 배치/그리드, 우(고정 패널)=선택 테이블 주문 패널
- 테이블 카드 표시: 테이블명 + 총금액 + 경과시간 + 인원
- 우측 패널: 테이블명/상태/경과/인원 + 주문 리스트 + 하단 "주문","결제" 버튼

## Food(푸드코트 주문)
- 본문 2-pane:
  - 좌 패널: 주문 리스트(상품명/수량/금액) + 하단 받는금액
  - 우 영역: 카테고리 탭 + 메뉴 카드 그리드
- 하단 고정 액션바: [반품/환불] [주문 보류] [결제 진행]
